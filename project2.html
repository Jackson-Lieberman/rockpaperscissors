Rock Paper Scissors

My first webpage coded in HTML and Java to play rock paper scissors.

Project Overview

Rock Paper Scissors started as a python project for my computer science class
I completed the project and decided to expand it to HTML.

Features

  • Play Rock Paper Scissors against a computer.
  • Saves scores of both sides.
  • Clean, vibrant page layout and UI.
  • Simple interface.

What I learned

  • This was my first time building a webpage in HTML
  • Got to experiment with CSS and Java
  • Learned funamentals of many new languages
  • Discovered a new skill that I enjoy!
